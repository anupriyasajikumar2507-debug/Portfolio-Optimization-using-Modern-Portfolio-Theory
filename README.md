# Portfolio-Optimization-using-Modern-Portfolio-Theory
This project applies Modern Portfolio Theory (Markowitz Mean-Variance Optimization) to construct an optimal equity portfolio using historical price data of 10 NSE-listed companies over five years. The objective is to maximize risk-adjusted returns while minimizing portfolio risk.

## Key Features
- Historical data collection using Yahoo Finance
- Daily and annualized return calculations
- Covariance and correlation analysis
- Mean-Variance portfolio optimization
- Efficient Frontier visualization
- Portfolio performance comparison

## Tools & Technologies
Python, Pandas, NumPy, SciPy, yfinance, Matplotlib, Microsoft Excel
